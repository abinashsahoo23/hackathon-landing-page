# Hackathon Landing Page

A responsive landing page for a hackathon event.

## Project Structure

- `index.html` - Main HTML file
- `style.css` - CSS styles
- `script.js` - JavaScript functionality
- `images/` - Directory containing image assets

## Getting Started

To view the landing page, simply open the `index.html` file in a web browser.

This is an ongoing project that will be updated with new features and improvements.
